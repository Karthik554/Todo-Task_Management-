# **To-Do List Mobile Application**

A simple and user-friendly To-Do List application built in **Java** using **Android Studio**. This app allows users to efficiently manage tasks by creating, updating, and organizing them with deadlines, statuses, and reminders.

---

## **Features**

- **Create Tasks**: Add new tasks with a title and description.
- **Set Deadlines**: Assign deadlines to tasks for better time management.
- **Update Tasks**: Edit task details, including title, description, deadline, status, and reminder time.
- **Set Status**: Categorize tasks as **Pending**, **In Progress**, or **Completed**.
- **Reminders**: Set specific times for task reminders using notifications.
- **Delete Tasks**: Remove completed or unnecessary tasks to keep your list organized.
